PHP: Les opérateurs - Manual

  • l’opérateur d’affectation : =
  • les opérateurs arithmétiques : + - * / % (voir modulo) ** (voir exponentiation)
  • les opérateurs booléens : && || xor and or ! (voir algèbre de Boole)
  • les opérateurs de comparaison : == != < > <= >=
  • les opérateurs d’incrémantation / décrémentation : ++ --
  • l’opérateur de concaténation de chaîne de caractères : .

Note : concaténer = chaîner, mettre bout à bout
